- 🚀 **GPT-4o-mini модель** - более точная классификация за меньшую цену

## ⚡ 3 шага для запуска

### 1. Демонстрация (без API ключа)
```bash
python demo_classifier.py
```
**Результат:** Классификация 6 популярных казахстанских программ с английскими ответами и SLUG полями.

### 2. Реальная классификация
```bash
# Установите API ключ OpenAI
set OPENAI_API_KEY=sk-ваш_ключ

# Запустите классификацию
python zero-shot-class.py
```

### 3. Проверка результатов
```bash
python check_demo_results.py
```
