## 🔧 Настройка для реального использования

### 1. Получение API ключа OpenAI

1. Зарегистрируйтесь на [OpenAI Platform](https://platform.openai.com/)
2. Создайте новый API ключ в разделе [API Keys](https://platform.openai.com/api-keys)
3. Скопируйте ключ (он начинается с `sk-`)

### 2. Настройка переменной окружения

**В Windows PowerShell:**
```powershell
$env:OPENAI_API_KEY="sk-ваш_ключ_здесь"
```

**В Windows CMD:**
```cmd
set OPENAI_API_KEY=sk-ваш_ключ_здесь
```

### 3. Запуск реальной классификации

```bash
python zero-shot-class.py
```
